Описание модуля "Архивный треккинг автотранспорта"

Данный модуль разработан для архивного хранения и предоставления данных о передвижении автотранспорта за запрашиваемый период в формате GPX. XSD схема, описывающая структуру применяемого формата представлена тут.

  • http://sarov.us.to:9902/trackviewer/xsd_scheme

    • Стандартная схема XSD GPX была расширена в части переопределения типа структур расширенных данных extensionsTypeTrk, extensionsTypeWpt, extensionsTypeSeg для объектов TypeTrk, TypeWpt, TypeSeg соответственно

    Описание методов модуля

    • http://sarov.us.to:9902/trackviewer/vehicle/show Get. Метод для получения списка отслеживаемых транспортных средств
    • http://sarov.us.to:9902/trackviewer/track/get?min_ts=0&max_ts=1643432956&vehicle_number=е008ен52 Get QueryParam. Метод возвращает GPX представление трека. В качестве параметров две метки времени, между которыми строится трек, и гос номер автомобиля
    • http://sarov.us.to:9902/trackviewer/track/get_json?min_ts=0&max_ts=1643432956&vehicle_number=е008ен52 Get QueryParam. Метод возвращает JSON представление трека. В качестве параметров две метки времени, между которыми строится трек, и гос номер автомобиля
    • http://sarov.us.to:9902/trackviewer/track/timeline Get QueryParam. Метод возвращает JSON, в котором указаны данные о пройденном пути, местах стоянок для каждого автомобиля за прошедшие 24 часа.
