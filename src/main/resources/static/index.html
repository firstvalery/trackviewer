Описание модуля "Архивный треккинг автотранспорта"

Данный модуль разработан для архивного хранения и предоставления данных о передвижении автотранспорта за запрашиваемый период в виде даннызх для построения трека, суточного, недельного таймлайна, отчетов в формате PDF

Описание методов модуля

  • http://sarov.us.to:9902/trackviewer/vehicle/show Get. Метод для получения списка отслеживаемых транспортных средств
  • http://sarov.us.to:9902/trackviewer/track/get_json?min_ts=0&max_ts=1643432956&vehicle_number=е008ен52 Get QueryParam. Метод возвращает JSON представление трека. В качестве параметров две метки времени, между которыми строится трек, и гос номер автомобиля
  • http://sarov.us.to:9902/trackviewer/track/timeline?ts=1544714152&type=ambulance Get QueryParam. Метод возвращает JSON, в котором указаны данные о пройденном пути, местах стоянок для каждого автомобиля типа type за сутки, которым принадлежит метка времени ts.
  • http://sarov.us.to:9902/trackviewer/track/timeline/get_week?ts=1544257200&type=ambulance Get QueryParam. Метод возвращает JSON, в котором указаны данные о пройденном пути, местах стоянок для каждого автомобиля типа type за неделю, где ts - метка времени последних суток искомой недели.
  • http://sarov.us.to:9902/trackviewer/track/report/commonPdf?ts_min=1544257200&ts_max=1554475000&type=ambulance Get QueryParam. Метод возвращает PDF документ, в котором сгенерирован отчет для транспортных средств типа type за период между двумя метками времени.
  • http://sarov.us.to:9902/trackviewer/track/report/specificPdf?ts_min=1544257200&ts_max=1554475000&number=н438мх152 Get QueryParam. Метод возвращает PDF документ, в котором сгенерирован отчет для транспортного средства с госномером number за период между двумя метками времени.
